## 4. 키움 API 한도 요약

- **API**: `ka10080` (분봉)
- **페이지당 반환**: 900개 고정
- **실제 커버 (실측)**: **약 300 분봉/일** (점심 휴장 11:30~13:00 반영)
- **응답 시간**: 평균 118ms
- **Rate limit**: 5 calls/sec (config)

### 종목당 1년치 수집
- 268일 × 300 분봉 = **80,400 분봉/종목**
- 페이지 수: **90회 호출/종목**
- API 시간: 90 ÷ 5 = **18초/종목** (순수 API)
- DB 병목 포함: **30초 ~ 5분/종목** (현 구현)

### 4/11~4/15 부분 수집
- **900 분봉/종목 = 1페이지**
- 60종목 × 1페이지 = **60 호출 = 12초 (API만)**
