# ADR-012: Universe 주간 자동 갱신

## 일자
2026-04-17

## 상태
ACCEPTED

## 결정

매주 월요일 07:30에 유니버스를 자동 갱신한다.

## 구현

engine_worker.py APScheduler 잡:
1. `scripts/generate_universe.py --min-atr 0.06` subprocess 실행
2. 변경 종목 식별 (추가/제거)
3. 신규 종목 분봉 수집 (batch_collector, 30일치)
4. active_strategies 재등록
5. WS 재구독
6. 전일 OHLCV 갱신
7. 텔레그램 알림 (변경 매트릭스)
8. 실패 시 기존 유니버스 유지 + 경고 알림

## 설정

- `notifications.universe_refresh: true` (토글 가능)
- 스케줄: `day_of_week=mon, hour=7, minute=30`
- `misfire_grace_time=600` (10분 유예)

## 배경

- 이전: 분기 1회 수동 재생성 권장
- 시장 변동성 변화에 대응하기 위해 주간 주기로 전환
- 월요일 07:30은 토큰 갱신(08:00) 전, 스크리닝(08:30) 전에 완료

## 실패 정책

- generate_universe.py 실패 → 기존 universe.yaml 유지
- 분봉 수집 실패 → 해당 종목만 skip, 나머지 정상 진행
- WS 재구독 실패 → 경고 로그, 다음 연결 시 자동 구독
