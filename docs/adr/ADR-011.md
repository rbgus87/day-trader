# ADR-011: Walk-Forward 검증 결과 — 전략 강건성 확인

## 일자
2026-04-16

## 상태
ACCEPTED

## 결정

Walk-Forward 검증 결과 전략 강건성이 확인되어 페이퍼 트레이딩 진입을 승인한다.

## 배경

ADR-010(Pure trailing + ATR 6%)으로 최적화한 전략이 과적합인지 검증 필요.

## 검증 방법

- 학습 기간: 2025-04-18 ~ 2026-01-31 (9.5개월)
- 검증 기간: 2026-02-01 ~ 2026-04-15 (2.5개월, 약세장 3월 포함)
- 파라미터: 학습 기간에서 확정된 값 그대로 검증 기간 적용

## 결과

| 구분 | PF | 거래수 | 승률 |
|------|-----|--------|------|
| 학습 | 5.11 | 225건 | — |
| 검증 | 4.05 | 54건 | — |
| 드롭 | -21% | | |

5가지 기준 전부 통과:
1. 검증 PF > 1.0 ✅ (4.05)
2. PF 드롭 < 30% ✅ (-21%)
3. 검증 거래수 > 30건 ✅ (54건)
4. 약세월 포함 ✅ (3월)
5. PF > 1 종목 비율 유지 ✅

## 한계

- 검증 기간 54건으로 통계적 유의성 약함
- 페이퍼 1개월이 진짜 검증
