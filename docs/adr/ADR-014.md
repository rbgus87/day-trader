# ADR-014: 분봉 자동 수집 (일일 15:35)

## 일자
2026-04-17

## 상태
ACCEPTED

## 결정

매일 평일 15:35에 유니버스 전체 종목의 당일 분봉을 자동 수집한다.

## 구현

engine_worker.py APScheduler 잡:
1. universe.yaml에서 전체 종목 로드
2. DataCollector.collect_minute_candles(ticker, days=1) 순차 호출
3. 성공/실패 종목 수 집계
4. 텔레그램 요약 알림

## 배경

- 분봉 실시간 DB 저장 미지원 (CandleBuilder는 in-memory 전용)
- 백테스트 데이터 연속성 + 페이퍼 검증에 분봉 필수
- 15:35은 장 마감(15:30) 직후, 일일 보고서(15:30)와 동시간대

## 설정

- `notifications.candle_collection: true` (토글 가능)
- 스케줄: `day_of_week=mon-fri, hour=15, minute=35`
- `misfire_grace_time=600`

## 공휴일

- APScheduler day_of_week=mon-fri로 주말 제외
- 공휴일은 KRX API가 빈 응답 → days=1로 수집 시 자연 skip (0건 저장)

## 소요 시간

- 드라이런 결과: 60종목 × ~2.8초/종목 ≈ 약 3분
- Rate limit 이슈 없음 (키움 REST API)
