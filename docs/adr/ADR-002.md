# ADR-002: 사이징 옵션 1 (1주 단위 비율) 채택

## 일자
2026-04-15

## 결정
백테스트와 라이브 모두 "1주 단위" 사이징 사용. 자본 기반 정교한 사이징은 검증된 baseline 위에 추후 추가.

## 배경
백테스트는 1주 기준 PnL 시뮬, 라이브는 자본 기반 risk 2% 사이징. PF 2.91은 자본 대비 수익률이 아닌 트레이드별 평균 수익률 지표. 두 시스템이 다른 사이징으로 비교 불가능.

## 대안

1. 1주 단위 비율 유지 (선택) → 단순, 백테스트 PF 의미 유지
2. 백테스트에 자본 사이징 추가 → 작업 큼, 새 PF 불확실
3. 동일 비중 단순화 → 잔여 자본 유휴 잔존
4. max_positions 확대 → 백테스트 재구현 필요

## 결정 이유
바탕이 있어야 추후 정교한 사이징 수정도 신뢰 가능.

## 상태
ACCEPTED
